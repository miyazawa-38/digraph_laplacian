### 信号を何にするか？

送信活動量が一番自然。
x_i = 頂点iが送信したメール総数
x = sum(A,2);
受信側を見るなら
x = sum(A,1).';

比較用の3つ
x_send = sum(A,2);  % 送信数
x_recv = sum(A,1).';  % 受信数
x_balance = x_send - x_recv;  % 送受信バランス

ただし、Aから直接作った信号だからグラフ構造と信号が強く結びついている。
独立した頂点属性でない。グラフとは別に与えられた信号を扱いたいなら、
時間窓ごとの活動量を信号にするほうが良い。

ex)
A = 全期間のメール数
x_i = 前半期間に頂点iが送信したメール

または
A = 前半期間のメール数
x_i = 前半期間に頂点iが送信したメール

にすると「ネットワーク構造から将来の活動量を見る」ような意味付けができる。
